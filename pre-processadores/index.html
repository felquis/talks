O que são pré-processadores?

Um pré-processador

É um programa que recebe um texto e efetua conversões léxicas nele.

O que seriam essas conversões léxicas?

Conversões léxicas é praticamente o mesmo que pegar um texto em português, escrito de forma informal, (por exemplo, com gírias e sem uma pontuação correta) e transforma-lo em um texto escrito de forma formal.

Essas conversões léxicas podem incluir macros de substituição, inclusão condicional, e inclusão de outros arquivos.

Macros

Segundo a Wikipédia (em outras palavras) macros são abstrações, ou seja, pequenas porções de código, que define como um padrão deve ser substituído por outro padrão.

Hum então em uma conversão léxica, o programa irá buscar um macro, e irá substituí-lo por outro padrão

Isso parece interessante.

Imaginando que você esteja usando Windows

Eu os apresento o Less CSS

O Less é um pré-processador de CSS, que funciona muito bem no Windows, ele é utilizado por grandes equipes de Front-Ends, como por exemplo do Twitter.

O Less, adiciona algumas features ao nosso velho e bom CSS, e é isso que faz toda a diferença. Por exemplo:

  • Podemos setar variáveis no nosso código;
  • Podemos incluir um arquivo CSS dentro de outro arquivo CSS
  • Criar mixins, que nada mais é que funções.
  • Regras aninhadas
  • Comentários em linha

Usando mixins

Usando um mixin


.transition(@valor: all 1s linear) {
    -webkit-transition: @valor;
    -moz-transition: @valor;
    -o-transition: @valor;
    -ms-transition: @valor;
    transition: @valor;
}

#seletor {
  .transition(all 1s linear);
}

Usando mixins

Resultado


#seletor {
  -webkit-transition: all 1s linear;
  -moz-transition: all 1s linear;
  -o-transition: all 1s linear;
  -ms-transition: all 1s linear;
  transition: all 1s linear;
}
