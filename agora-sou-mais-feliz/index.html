Vamos começar instalando

Se você já tem o Ruby instalado, apenas rode "gem install compass"

Como instalar o Compass

Crie um projeto com Compass


compass create --sass-dir "sass" --css-dir "css" --javascripts-dir "js" --images-dir "img"

Comandos básicos

compass watch

e

compass compile

#1 coisa que me torna mais feliz

Por que eu...

Posso concatenar arquivos

Antes

Eu tinha que trabalhar em arquivos muito grandes 1, 2, 3 mil linhas de CSS

Por que é importante evitar usar o @import do CSS

Hoje

Eu posso usar @import livremente por que o pré-processador concatena os arquivos para mim.

Por exemplo se no arquivo.scss


    @import "arquivo2.scss";
    @import "arquivo3.scss";

O arquivo.scss vai conter os CSS dos dois sendo 'chamados' dentro dele.

#2 coisa que me torna mais feliz

Por que eu...

Escrevo menos código

Antes eu fazia assim:


.item-wrap .item {
    -webkit-transform: rotate(10deg);
    -moz-transform: rotate(10deg);
    -ms-transform: rotate(10deg);
    -o-transform: rotate(10deg);
    transform: rotate(10deg);
}

Hoje eu faço assim:


.item-wrap .item {
    @include transform(rotate(10deg));
}

Isso se chama mixin

#3 coisa que me faz mais feliz

São as regras aninhadas

Antes eu fazia assim:


    .item-wrap {
        overflow: hidden;
        margin: 0;
    }

    .item-wrap .item {
        float: left;
        padding: 10px 10px 10px 20px;
        text-decoration: none;
    }

    .item-wrap .item:hover {
        text-decoration: underline;
    }

Hoje eu faço assim:


    .item-wrap {
        overflow: hidden;
        margin: 0;

        .item {
            float: left;
            padding: 10px 10px 10px 20px;
            text-decoration: none;

            &:hover {
                text-decoration: underline;
            }
        }
    }

#4 coisa que me faz mais feliz

As variáveis

Antes eu fazia assim:


    .algum .seletor {
        color: #19cade;
    }

    /* Depois de 70 linhas */

    .outro .seletor {
        color: #19cade;
    }

    /* Depois de 100 linhas */

    .mais-um .seletor {
        color: #19cade;
    }

Hoje eu faço assim:


    $cor_geral : #19cade;

    .algum .seletor {
        color: $cor_geral;
    }

    /* Depois de 70 linhas */

    .outro .seletor {
        color: $cor_geral;
    }

    /* Depois de 100 linhas */

    .mais-um .seletor {
        color: $cor_geral;
    }

#5 coisa que me torna mais feliz

#5 coisa que me torna mais feliz

Os mixins

Criando um mixin


@mixin transform($params) {
    -webkit-transform: $params;
    -moz-transform: $params;
    -ms-transform: $params;
    -o-transform: $params;
    transform: $params;
}

Usando um mixin


body {
    @include transform(rotate(10deg));
}

#6 coisa que me torna mais feliz

Sprite generator

Antes

Eu criava uma imagem no Photoshop encaixando todas as imagens do meu sprite

E para coloca-los na posição certa no CSS era muito difícil

Hoje

Como eu uso o sprite generator minha vida é bem mais fácil. 

@import "icones/*.png";

.item {
    @include icones-sprite('icon1');
    display: block;
    width: icones-sprite-width('icon1');
    height: icones-sprite-height('icon1');
}

E a mágica acontece


.icones-sprite, .item {
  background: url('/img/icones-s3029a34d13.png') no-repeat;
}

.item {
  background-position: 0 0;
  display: block;
  width: 48px;
  height: 48px;
}
