Agora que esta tudo pronto

#1 coisa que me torna mais feliz

Por que eu...

Posso concatenar arquivos

No arquivo teste.scss eu coloquei


    @import "arquivo1.scss";
    @import "arquivo2.scss";

Agora o arquivo teste.scss possui o conteúdo dos arquivos arquivo1.scss e arquivo2.scss

Por inquanto ignorem os arquivos com .scss

#2 coisa que me torna mais feliz

Por que eu...

Escrevo menos código

Antes eu fazia assim:


.item-wrap .item {
    -webkit-transform: rotate(10deg);
    -moz-transform: rotate(10deg);
    -ms-transform: rotate(10deg);
    -o-transform: rotate(10deg);
    transform: rotate(10deg);
}

Hoje eu faço assim:


.item-wrap .item {
    @include transform(rotate(10deg));
}

Isso se chama mixin

#3 coisa que me faz mais feliz

Regras aninhadas

Antes eu fazia assim:


    .item-wrap {
        overflow: hidden;
        margin: 0;
    }

    .item-wrap .item {
        float: left;
        padding: 10px 10px 10px 20px;
        text-decoration: none;
    }

    .item-wrap .item:hover {
        text-decoration: underline;
    }

Hoje eu faço assim:


    .item-wrap {
        overflow: hidden;
        margin: 0;

        .item {
            float: left;
            padding: 10px 10px 10px 20px;
            text-decoration: none;

            &:hover {
                text-decoration: underline;
            }
        }
    }

#4 coisa que me faz mais feliz

Variáveis

Antes eu fazia assim:


    .algum .seletor {
        color: #19cade;
    }

    /* Depois de 70 linhas */

    .outro .seletor {
        color: #19cade;
    }

    /* Depois de 100 linhas */

    .mais-um .seletor {
        color: #19cade;
    }

Hoje eu faço assim:


    $cor_geral : "#19cade";

    .algum .seletor {
        color: $cor_geral;
    }

    /* Depois de 70 linhas */

    .outro .seletor {
        color: $cor_geral;
    }

    /* Depois de 100 linhas */

    .mais-um .seletor {
        color: $cor_geral;
    }

#5 coisa que me torna mais feliz

Comentário em linha

Quem foi o santo que criou apenas comentário em bloco para o CSS?

Antes eu fazia assim:


    /* Cores do topo */
    .topo-de-dia {
        background: white;
    }

    .topo-de-noite {
        background: black;
    }

Hoje eu faço assim:


    // Cores do topo
    .topo-de-dia {
        background: white;
    }

    .topo-de-noite {
        background: black;
    }
